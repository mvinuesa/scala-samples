# scala-samples
Scala samples

Scala samples to learn basic concepts in Scala language

#### Compile it

In *.scala folder

  * *sbt compile* 
  * *scalac 'file'.scala*

#### Run it

  * *sbt run* 
  * *sbt console*
    *scala> 'package'.file'.main(Array())*
  * *scala 'file'*
  
#### Example: 
  
  * *scala> ship.ShipMain.main(Array())*
